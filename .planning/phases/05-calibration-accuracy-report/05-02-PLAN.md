**Create scripts/generate_accuracy_report.py** following the generate_cross_validation.py pattern exactly:

1. **Preamble:** CadQuery mock, REPO_ROOT, sys.path (identical to generate_cross_validation.py)

2. **grade_metric(computed, reference, tolerance_abs, tolerance_pct) -> tuple[str, float, float]:**
   - Compute error_abs = abs(computed - reference)
   - Compute error_pct = abs(computed - reference) / abs(reference) * 100.0 if reference != 0 else 0.0
   - Use tolerance_abs if not None, else tolerance_pct, else return "UNGRADED"
   - PASS: error <= tolerance, MARGINAL: error <= 2*tolerance, FAIL: error > 2*tolerance
   - Return (grade, error_abs, error_pct)

3. **validate_sources(report, ref_data) -> None:**
   - Build valid source set from reference_data.json keys (aircraft_specs + airfoil_data) prefixed with "reference_data.json:" plus "vspaero_native"
   - For each metric in report["metrics"]: raise ValueError if source contains "physics_baseline.json" or source not in valid set

4. **load_vspaero_provenance(data_dir) -> dict:**
   - Load from vspaero_native_polars.json: vsp_version, timestamp -> run_timestamp, solver_settings
   - Return fallback dict if file missing

5. **collect_metrics(ref_data, config_module, engine) -> list[dict]:**
   Collect these metrics (per locked decision — ALL validated metrics):

   **Stability (from PhysicsEngine):**
   - neutral_point_fs: engine.calculate_cg_envelope().neutral_point -> to_published_datum(), ref=108.0, source="reference_data.json:aircraft_specs.neutral_point_fs"
   - cg_range_fwd_fs: metrics.cg_range_fwd -> to_published_datum(), ref=99.0, source="reference_data.json:aircraft_specs.cg_range_fwd_fs"
   - cg_range_aft_fs: metrics.cg_range_aft -> to_published_datum(), ref=104.0, source="reference_data.json:aircraft_specs.cg_range_aft_fs"
   - static_margin_pct: metrics.static_margin * 100, ref from reference_data.json static_margin_pct, source="reference_data.json:aircraft_specs.static_margin_pct"

   **Performance:**
   - stall_speed_ktas: first-principles formula (same as test_precision_validation.py using published areas 94.2+15.6 sqft), ref=56, source="reference_data.json:aircraft_specs.stall_speed_ktas"
   - max_gross_weight_lb: config.flight_condition.gross_weight_lb, ref=1425, source="reference_data.json:aircraft_specs.max_gross_weight_lb"
   - empty_weight_lb: sum of config structural weights (or config.structural_weights total if available), ref=850, source="reference_data.json:aircraft_specs.empty_weight_lb"

   **Airfoil (from config):**
   - canard_clmax: config.aero_limits.canard_clmax, ref=1.35, source="reference_data.json:airfoil_data.roncz_r1145ms.cl_max"
   - wing_clmax: config.aero_limits.wing_clmax, ref=1.45, source="reference_data.json:airfoil_data.eppler_1230.cl_max"
   - canard_alpha_0l_deg: config.aero_limits.canard_alpha_0L, ref=-3.0, source="reference_data.json:airfoil_data.roncz_r1145ms.alpha_zero_lift_deg"
   - wing_alpha_0l_deg: config.aero_limits.wing_alpha_0L, ref=-2.0, source="reference_data.json:airfoil_data.eppler_1230.alpha_zero_lift_deg"

   **Geometry:**
   - wing_area_sqft: config.geometry.wing_area (converted to sqft if in sq-in), ref=94.2, source="reference_data.json:aircraft_specs.wing_area_sqft", add convention_note field about full trapezoidal vs RAF semi-panel convention

   For each metric: read tolerance_abs and tolerance_pct from reference_data.json. Use airfoil tolerances of 0.05 for CLmax and 0.5 for alpha_0L (matching test_precision_validation.py values). For max_gross_weight_lb (tolerance_abs=null), use exact match logic (tolerance_abs=0).

6. **build_report(metrics, vspaero_provenance) -> dict:**
   - metadata: generated (ISO timestamp), config_version (from config.version if available), vspaero_provenance, traceability note
   - summary: count PASS/MARGINAL/FAIL/UNGRADED
   - metrics: list of metric dicts with metric_id, description, computed, reference, tolerance_abs, tolerance_pct, error_abs, error_pct, grade, source, units

7. **generate_accuracy_report(output_path=None) -> Path:**
   - Main entry point callable from main.py
   - Default output_path = REPO_ROOT / "data" / "validation" / "accuracy_report.json"
   - Load ref_data, create PhysicsEngine, collect metrics, grade, validate sources, write JSON
   - Return the output path

8. **main():**
   - Call generate_accuracy_report() and print summary to stdout
