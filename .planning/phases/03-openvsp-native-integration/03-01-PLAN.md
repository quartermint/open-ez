Add missing OpenVSP geometry parameters to aircraft config and create an installation script for OpenVSP Python bindings on macOS ARM64.

  <action>
1. In `config/aircraft_config.py`, add these fields to the `GeometricParams` dataclass under a new `# === OPENVSP GEOMETRY (positions for 3D model) ===` section, placed after the existing fuselage section:

   - `wing_le_wl: float = 0.0` — Wing LE waterline (Z position in inches, relative to longerons. Long-EZ wing is roughly at longeron level.)
   - `canard_le_wl: float = 12.0` — Canard LE waterline (canard is ~12" above longerons on Long-EZ)
   - `winglet_height: float = 16.0` — Winglet vertical span in inches (Long-EZ winglets are ~16" tall)
   - `winglet_root_chord: float = 20.0` — Winglet root chord in inches (at wing tip junction)
   - `winglet_tip_chord: float = 12.0` — Winglet tip chord in inches
   - `fuselage_length: float = 214.0` — Total fuselage length (= fs_tail, nose to tail terminus)

2. Add computed properties that alias FS values for OpenVSP compatibility:
   ```python
   @property
   def wing_le_fs(self) -> float:
       """Wing LE fuselage station (alias for fs_wing_le, used by OpenVSP geometry builder)."""
       return self.fs_wing_le

   @property
   def canard_le_fs(self) -> float:
       """Canard LE fuselage station (alias for fs_canard_le, used by OpenVSP geometry builder)."""
       return self.fs_canard_le
   ```

3. Create `scripts/install_openvsp.sh`:
   - Download OpenVSP 3.48.2 macOS ARM64 Python 3.13 ZIP from https://openvsp.org/download.php
   - Extract to `~/.local/openvsp/` (or configurable location)
   - Create a Python venv with Python 3.13 (since the MacBook has 3.14 and OpenVSP bundles 3.13 bindings)
   - Detect the `python/` subdirectory in the OpenVSP bundle that contains the `.so` bindings
   - Create a `.pth` file in the project venv's site-packages pointing to the OpenVSP python directory
   - Run a smoke test: `python -c "import openvsp as vsp; print(vsp.GetVSPVersion())"`
   - Print instructions for activating the venv
   - IMPORTANT: The CONTEXT.md says "conda" but conda-forge does NOT have openvsp. The macOS app bundle is the only distribution. The script should note this.

4. Add a comment in requirements.txt noting the install script path:
   Update the existing commented `# openvsp>=3.35.0` line to:
   `# openvsp 3.48.2 — installed via scripts/install_openvsp.sh (not available on pip/conda)`

Note on dimension values: The winglet, waterline, and fuselage dimensions are derived from Long-EZ plans (Rutan plans Chapter 19 winglets, general arrangement drawings). These are approximate values suitable for VLM analysis — exact values are less critical since VSPAERO VLM is mesh-resolution limited anyway.
