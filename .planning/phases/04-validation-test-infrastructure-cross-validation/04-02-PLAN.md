  <files>scripts/generate_cross_validation.py, data/validation/vspaero_native_polars.json, data/validation/surrogate_cross_validation.json</files>
  <action>
    Create `scripts/generate_cross_validation.py` with three functions:

    1. `regenerate_native(data_dir: Path) -> dict`:
       - Instantiate `VSPIntegration()`
       - Assert `bridge.has_vsp` — fail with clear message if OpenVSP not available
       - Call `bridge._run_native_sweep((-4, 14, 19), polar_output=data_dir / "vspaero_native_polars.json")`
       - Assert `"mock" not in result["vsp_version"].lower()` — fail if mock data returned
       - Return the result dict
       - Print summary: version, point count, CL range

    2. `get_surrogate_polars(alphas: list[float]) -> list[dict]`:
       - Instantiate `OpenVSPAdapter()`
       - Call `adapter.run_vspaero(alphas)` with the SAME 19-point alpha array used by native sweep
       - Convert AeroPolar objects to dicts: `{"alpha_deg": p.alpha_deg, "cl": p.cl, "cd": p.cd, "cm": p.cm}`
       - Return list of dicts

    3. `build_discrepancy_table(native_points: list, surrogate_points: list) -> dict`:
       - For each matching alpha point, compute delta (native - surrogate) for CL, CD, CM
       - Compute per-metric summary: mean_delta, rms_delta, max_abs_delta
       - Return structured dict:
         ```python
         {
             "metadata": {
                 "generated": datetime.utcnow().isoformat(),
                 "native_source": "VSPAERO VLM (real)",
                 "surrogate_source": "OpenVSPAdapter (lifting-line + Viterna)",
                 "alpha_count": len(native_points),
                 "note": "Measure-only. No pass/fail thresholds. Phase 5 decides calibration."
             },
             "alpha_deg": [...],
             "comparison": [
                 {"alpha_deg": float, "cl_native": float, "cl_surrogate": float, "cl_delta": float,
                  "cd_native": float, "cd_surrogate": float, "cd_delta": float,
                  "cm_native": float, "cm_surrogate": float, "cm_delta": float},
                 ...
             ],
             "summary": {
                 "cl": {"mean_delta": float, "rms_delta": float, "max_abs_delta": float},
                 "cd": {"mean_delta": float, "rms_delta": float, "max_abs_delta": float},
                 "cm": {"mean_delta": float, "rms_delta": float, "max_abs_delta": float}
             }
         }
         ```

    4. `main()`:
       - Set up paths: `REPO_ROOT = Path(__file__).resolve().parents[1]`, `DATA_DIR = REPO_ROOT / "data" / "validation"`
       - Call `regenerate_native(DATA_DIR)` — this writes `vspaero_native_polars.json`
       - Build alpha array: `alphas = [float(a) for a in range(-4, 15)]` (19 points: -4 to 14)
       - Call `get_surrogate_polars(alphas)`
       - Call `build_discrepancy_table(native_points, surrogate_points)`
       - Write result to `DATA_DIR / "surrogate_cross_validation.json"` with `json.dump(indent=2)`
       - Print summary table to stdout

    5. Add `if __name__ == "__main__": main()` guard.

    6. IMPORTANT: The CadQuery mock is needed before importing core modules. Add `sys.modules.setdefault("cadquery", MagicMock())` and `sys.modules.setdefault("OCP", MagicMock())` at the top of the script before any core imports.

    7. After creating the script, run it under Python 3.13:
       ```bash
       cd /Users/ryanstern/open-ez && python3.13 scripts/generate_cross_validation.py
       ```
       This will:
       - Regenerate `data/validation/vspaero_native_polars.json` with real VSPAERO data
       - Create `data/validation/surrogate_cross_validation.json` with discrepancy table

    8. After running, verify `vspaero_native_polars.json` no longer contains "mock" in vsp_version.
